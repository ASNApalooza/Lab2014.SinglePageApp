Part 0: Start

This part of the tutorial contains instructions for installing the empty project template. You should already have Visual Studio 2012 and ASNA Visual RPG for .NET installed.

Visual Studio can be downloaded from here.
ASNA Visual RPG for .NET is available at the ASNA Developer Network.
Download

Installation

Download the project template using the button above, then move the zip file to your projects folder.

Unzip the file and rename the extracted folder from Lab2014.SinglePageApp-part-0-start to SinglePageApp.

You should now have a SinglePageApp folder that contains the following structure: 

SinglePageApp
|   .gitignore
|   favicon.ico
|   README.md
|   Web.config
|   Web.Debug.config
|
\---App_Data
        Application.mdf
        Application_log.ldf
        ContactSeeder.zip
        dbo.Contacts.sql

The App_Data folder contains an SQL database as an alternative to the local DataGate database. The database layer is covered in part 2 of the tutorial.

The .gitignore and README.md files are used by the Git repository.

The Web.config file is mostly empty other than a connection string for the SQL database:

Web.config 

<?xml version="1.0"?>
<!--
  For more information on how to configure your ASP.NET application, please visit
  http://go.microsoft.com/fwlink/?LinkId=169433
  -->
<configuration>
  <connectionStrings>
    <add name="Application" providerName="System.Data.SqlClient" connectionString="Data Source=(LocalDB)\v11.0;AttachDbFilename=|DataDirectory|\Application.mdf;Integrated Security=True" />
  </connectionStrings>
  <system.web>
    <compilation debug="true" targetFramework="4.5" />
  </system.web>
  <system.webServer>
    <modules runAllManagedModulesForAllRequests="true" />
  </system.webServer>
</configuration>

The favicon.ico file provides a simple bookmark icon for the project.

Open Visual Studio and then click “FILE” then “Open Web Site...” on the menu bar. Browse to the SinglePageApp folder and click “OPEN”.

Once the website loads, we can move on and get started. Click the Next button below to continue.
