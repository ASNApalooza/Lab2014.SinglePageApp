Overview

This repository contains example code for the ASNApalooza 2014 Lab:
“Build a Single-Page App with an AVR Web API.”

This code is provided for learning purposes only and is not meant for production use. This tutorial application does not include authentication, paging, or extensive error handling.

The documentation site includes a tutorial, downloads, and the introductory slides. You may also browse the code snippets used in the live lab, and the Backbone snippets for the JavaScript portion of the project.
